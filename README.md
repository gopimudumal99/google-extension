## Run Locally

Clone the project

```bash
  git clone https://github.com/gopimudumal99/google-extension.git
```

Go to the google-extension

```bash
  cd google-extension
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
